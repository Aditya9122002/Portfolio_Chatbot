# 🤖 Aditya Agarwal — AI Portfolio Assistant

An AI-powered portfolio chatbot that answers questions 
about my skills, projects, experience and more.

🔗 **Live Demo:** https://portfoliochatbot-84ja9qnxuwdshyp2aktdtt.streamlit.app/

## Features
- 6 quick question buttons for mobile users
- Conversation memory
- Powered by Groq + LLaMA3
- 100% free to run

## Tech Stack
Python · Groq API · Streamlit · LLaMA3

## Run Locally
```bash
git clone https://github.com/Aditya9122002/Portfolio_Chatbot
pip install -r requirements.txt
cp .env.example .env  # fill in your details
streamlit run app.py
```
